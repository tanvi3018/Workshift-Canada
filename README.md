# WorkShift Canada 🍁

**An interactive bilingual data dashboard analyzing remote work trends across Canada, 2020–2025.**

Built by Tanvi Gera · Business Analyst · ESDC Canada

---

## What this dashboard covers

- **17 Canadian industries** — WFH capacity, COVID adoption, and actual 2025 remote rates
- **13 provinces and territories** — choropleth map with economic cluster analysis
- **9 major cities** — hybrid and remote job posting data
- **Public vs private sector scenario** — where remote work is headed by 2027
- **Bilingual** — full English and French throughout

## Data sources

- Statistics Canada · Table 33-10-0228-01 · Open Government Licence
- Statistics Canada Labour Force Survey 2020–2025
- Robert Half Canada Salary Guide Q4 2025
- Treasury Board of Canada Secretariat

## Run locally

```bash
pip install -r requirements.txt
streamlit run remote_work_app.py
```

## Tech stack

Python · Streamlit · Plotly · Pandas · Carto tiles (no API token required)
